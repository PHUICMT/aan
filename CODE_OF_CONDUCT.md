# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful. Harassment, personal attacks, and discriminatory
language are not tolerated.

## Project-specific rules

- Aan is a library manager. Do not ask for, share, or link to copyrighted
  content, scanlations, or pirate sites in issues, discussions, or PRs.
  Violations will be removed and may result in a ban.
- Keep discussions focused on the software, not on what users read with it.

## Reporting

Open a private security advisory on GitHub or email the maintainer.
