# Aan

A personal library manager for manga and novels.

*Aan* (อ่าน) means "read" in Thai.

## Status

Pre-alpha. Not ready for use yet.

## Planned features

- Import PDF, EPUB, CBZ, CBR, image folders, plain text
- Reader for both manga (paged / spread / continuous) and novels
- Library with tags, smart collections, reading stats
- Watch folder for auto-import
- Multi-language: EN, TH, JA, ZH-CN at launch
- 100% free, all features, forever — supported by donations

## Stack

Tauri 2 · Svelte 5 · Rust · SQLite
